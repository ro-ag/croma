# Raw comparator baseline (pre-triage)

Generated by `tools/music21_polars_corpus_compare.py` after the six
match-forcing normalizations were stripped (commit `381a1d3`). This is the
**honest** baseline: the comparator reports raw structural differences and forces
no matches. Equivalences and abc2xml artifacts are adjudicated explicitly in the
worklist, not hidden here.

## Headline (10,000-tune corpus)

| Metric | Value |
| --- | ---: |
| Files compared | 9,935 |
| **Structural matches (whitelist)** | **8,583** |
| **Structural mismatches (worklist)** | **1,352** |
| Croma export failures (not compared) | 65 |
| Total mismatch rows | 102,979 |

For reference, the previous *normalized* comparator reported 9,411 matches / 524
mismatches — the six normalizations were silently converting **828** files into
matches.

normalization — one equivalence class to adjudicate once in the worklist.

## Files

- **Whitelist** — `whitelist.csv` (this directory's sibling): the 8,583
  raw-match filenames. Regression baseline: a croma change that breaks any of
  these is a regression.
- **Worklist** — the 1,352 mismatched files, triaged one
  at a time by the `abc-divergence-investigator` subagent via the
  `divergence-triage` skill into either a croma-bug fix or a reasoned
  `dropped.csv` entry.

## Reproduce

```sh
uv run python tools/music21_polars_corpus_compare.py \
  --results-jsonl <croma-export-results.jsonl> \
  --croma-xml-root <croma-xml-root> --reference-root <reference-musicxml-root> \
  --report report.json \
  --whitelist-csv docs/comparison/abc2xml-divergences/whitelist.csv \
  --dropped-csv docs/comparison/abc2xml-divergences/dropped.csv
```
