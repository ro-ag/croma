tree-sitter-abc — web/WASM highlighting demo

The same tree-sitter-abc grammar that drives Zed and croma's GPUI app, compiled to WebAssembly and run in the browser via web-tree-sitter. Edit the ABC on the left; the right pane is highlighted live by queries/highlights.scm — no server-side processing, no build step (the grammar wasm is committed), and the runtime is vendored so it works offline.
